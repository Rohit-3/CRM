```tsx

  <CardHeader className="pb-4">
    <CardTitle className="text-lg">{Title}</CardTitle>

      {Subtitle}

    {/* Content */}
  </CardContent>
```

### Key Styling Rules

1. **CardHeader**: Always use `pb-4` for consistent bottom padding
2. **CardTitle**: Always use `text-lg` for uniform title size
3. **CardContent**: Always use `pt-0` to remove top padding
4. **Subtitle**: Always use `text-sm text-muted-foreground mt-1`
